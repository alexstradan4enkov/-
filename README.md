Функция findMin

Функция findMin находит минимальное значение в переданном векторе целых чисел.

Параметры:

· const vector<int>& numbers - вектор целых чисел

Возвращаемое значение:

· int - минимальное значение или INT_MAX если вектор пуст

Пример использования:
vector<int> numbers = {1, 5, 3, 9, 2};
int minValue = findMin(numbers); // 1

Основная программа

Программа запрашивает у пользователя:

1. Количество элементов в массиве
2. Значения каждого элемента
3. Выводит минимальное значение
