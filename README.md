# Проект для поиска максимального значения на C++

## Функция findMax

Функция findMax находит максимальное значение в переданном векторе целых чисел.

### Параметры:
- const vector<int>& numbers - вектор целых чисел

### Возвращаемое значение:
- int - максимальное значение или INT_MIN если вектор пуст

### Пример использования:
vector<int> numbers = {1, 5, 3, 9, 2};
int maxValue = findMax(numbers); // 9
